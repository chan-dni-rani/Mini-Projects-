## 📋 Overview

A beautiful and functional calculator built with pure vanilla JavaScript. It features basic arithmetic operations, quick clear/delete actions, and a modern gradient design that works flawlessly across all devices. Perfect for quick calculations right in your browser! 💜

---

## ✨ Features

- ➕ **Basic Operations** - Addition, subtraction, multiplication, division, and percentage
- 🧹 **Clear All (AC)** - Reset calculator instantly
- ⌫ **Delete Last (DEL)** - Remove the last digit
- 🔢 **Decimal Support** - Handle decimal numbers with ease
- ⚡ **Double-Zero Input** - Quick entry for large numbers
- 📱 **Responsive Design** - Centered layout that works on all screen sizes
- 🎨 **Modern Gradient UI** - Eye-catching purple gradient background

## 🛠️ Tech Stack

| Technology | Purpose |
